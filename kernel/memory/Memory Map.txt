#=========================#========#==================================================================#
|          Range          |  Size  |                           Description                            |
#=========================#========#==================================================================#
| 0x00000000 - 0x0001FFFF | 128KB  | Reserved for low-level hardware, like IVT                        |
| 0x00020000 - 0x001FFFFF | 1920KB | For kernel codes                                                 |
| 0x00200000 - ...        |  ???   | Free memory                                                      |
#=========================#========#==================================================================#
